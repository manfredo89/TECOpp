Brief

TECO++ is a C++ translation of the Fortran TECO model.
TECO is a Carbon transfer model introduced by Y. Luo and J. Reynolds (1999).
Please cite original reference when using this algorithm.
https://esajournals.onlinelibrary.wiley.com/doi/10.1890/0012-9658%281999%29080%5B1568%3AVOEFCE%5D2.0.CO%3B2


The equation describing the carbon transfer is the matrix representation of

\( \frac{dX(t)}{dt} = BI(t) - A \xi KX(t) \)


To compile link with Armadillo libraries (http://arma.sourceforge.net/)

Test run for 10000 years with different compilers for the same code:
TECO++ (icpc -fast) -> 9.112 s
TECO++ (clang++ -Ofast) -> 8.689 s
TECO (ifort -fast) -> 6.107 s

By optimizing code (avoid repeated computation of AK_inv which is constant):
TECO++ (clang++ -Ofast) -> 2.33873 s
